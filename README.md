# motw-sheet
Digital character sheet for Monster of the Week

| Task                               | Command            |
|------------------------------------|--------------------|
| Project Setup                      | `npm install`      |
| Compile and Hot-Reload for Dev     | `npm run dev`      |
| Compile and Minify for Production  | `npm run build`    |
| Lint with ESLint                   | `npm run lint`     |
